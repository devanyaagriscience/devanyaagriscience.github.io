# Mustard DAS 4747

## Overview
**Mustard DAS 4747** is a high-yielding oilseed variety. It is a favorite among farmers for its vigorous growth and among millers for its high oil recovery percentage.

## Key Features
- **High Oil Content**: 40-42% oil content ensures better profitability.
- **Bold Seeds**: Large, dark brown seeds.
- **Frost Tolerant**: Genetic traits that offer protection against cold waves.

## Technical Specifications
| Characteristic | Description |
| :--- | :--- |
| **Maturity** | 125-135 Days |
| **Plant Height** | 160-180 cm |
| **Branching** | Profuse branching from base |
| **Season** | Rabi |

## Cultivation Tips
Sow in rows to facilitate intercultural operations. Apply sulfur-based fertilizers to enhance oil content. Protection against aphids at the flowering stage is recommended.
