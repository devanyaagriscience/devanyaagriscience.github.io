# Onion Fursungi

## Overview
**Onion Fursungi** is a market-leading selection known for its exceptional storage capabilities. It allows farmers to store their harvest for months and sell when market rates are favorable.

## Key Features
- **Long Storage Life**: Can be stored for 6-8 months with minimal weight loss or sprouting.
- **Uniform Bulbs**: Produces consistently sized, globular bulbs.
- **Attractive Color**: Deep bright red color that appeals to consumers.

## Technical Specifications
| Characteristic | Description |
| :--- | :--- |
| **Maturity** | 110-120 Days after transplanting |
| **Bulb Weight** | 80-100 grams |
| **Shape** | Round / Globular |
| **Season** | Late Kharif / Rabi |

## Cultivation Tips
Requires well-drained soil rich in organic matter. Avoid excessive nitrogen late in the season to improve storage quality. Cure the bulbs properly in the shade before storage.
