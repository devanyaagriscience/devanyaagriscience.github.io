# Paddy DAS 6060

## Overview
**Paddy DAS 6060** is a robust short-to-medium duration variety. It is specifically famous for its non-lodging character, which protects the crop from heavy rains and winds near harvest time.

## Key Features
- **Non-Lodging Stem**: Strong stem architecture prevents falling, minimizing crop loss.
- **Early Maturity**: Ideal for double cropping patterns as it matures quickly.
- **Good Milling Quality**: High recovery rate during milling with minimal broken grains.

## Technical Specifications
| Characteristic | Description |
| :--- | :--- |
| **Duration** | 110-115 Days |
| **Plant Height** | 90-100 cm |
| **Grain Type** | Long Bold |
| **Sowing Season** | Kharif / Rabi |

## Cultivation Tips
This variety covers the ground quickly, suppressing weeds naturally. Maintain a shallow layer of water after transplantation and increase it as the crop grows.
