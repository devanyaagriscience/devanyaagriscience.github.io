# Paddy DAS 8040

## Overview
**Paddy DAS 8040** is built for resilience, combining disease resistance with high output. It performs exceptionally well in irrigated belts, offering a dense panicle structure that translates to heavier weighing bags.

## Key Features
- **Disease Resistance**: Strong resistance to blast and bacterial leaf blight.
- **Heavy Panicles**: Long and dense panicles with high grain filling.
- **High Test Weight**: Grains are heavy, resulting in more weight per acre.

## Technical Specifications
| Characteristic | Description |
| :--- | :--- |
| **Duration** | 135-140 Days |
| **Plant Height** | 100-110 cm |
| **Grain Type** | Bold |
| **Sowing Season** | Kharif |

## Cultivation Tips
Best suited for low-lying areas with good water availability. Requires minimal pesticide sprays due to inherent resistance.
