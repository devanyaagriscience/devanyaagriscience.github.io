# Pea Sweet Green

## Overview
**Pea Sweet Green** is designed for the freshest, sweetest garden peas. This variety produces long, well-filled pods with tender green peas that are perfect for the fresh market.

## Key Features
- **Sweet & Tender**: High sugar content makes it very palatable.
- **Long Pods**: Each pod contains 8-10 grains.
- **Market Favorite**: High demand in vegetable markets.

## Technical Specifications
| Characteristic | Description |
| :--- | :--- |
| **Maturity** | 60-65 Days |
| **Plant Height** | 50-60 cm |
| **Pod type** | Dark Green |
| **Season** | Rabi |

## Cultivation Tips
Needs cool weather for best pod setting. Staking may be required for support if vegetative growth is vigorous.
