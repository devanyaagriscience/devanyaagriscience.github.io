# Urad PU-31

## Overview
**Urad PU-31** is a trusted heavy-yielder in the black gram category. It is resistant to Yellow Mosaic Virus (YMV) and produces bold, black seeds, making it a robust cropper.

## Key Features
- **YMV Resistant**: Highly resistant to Yellow Mosaic Virus.
- **Bold Seeds**: Large seed size fetches better prices.
- **High Protein**: Excellent nutritional profile.

## Technical Specifications
| Characteristic | Description |
| :--- | :--- |
| **Maturity** | 75-80 Days |
| **Plant Height** | 45-50 cm |
| **Pod Color** | Black |
| **Season** | Kharif / Spring |

## Cultivation Tips
Can be grown as a catch crop between major cereal crops. Requires minimal nitrogen as it is a legume.
